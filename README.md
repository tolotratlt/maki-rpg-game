# Captain Clown Nose RPG

A small 2D pixel RPG prototype built with `@tialops/maki`.

## Run the game

```bash
npm install
npm run dev
```

If Vite does not open the browser automatically, open `http://127.0.0.1:5173/`.

## Project structure

```text
assets/
scenes/
game.js
index.html
maki.config.js
vite.config.js
```

## Controls

- `Arrow keys`: move Captain Clown Nose
